## Instalacion
```bash
git clone https://github.com/alialucas7/st/
```
```bash
cd st
```
```bash
sudo make clean install
```
## Modificaciones para el PS1
mover el archivo '.bashrc' a la carpeta del usuario 

```bash
mv .bashrc ~/
```
